### 下载安装

* 下载源码:

```shell
git clone git@github.com:wengAlan/Proxy_pool.git

或者直接到https://github.com/wengAlan/Proxy_pool 下载zip文件
```

* 安装依赖:

```shell
pip install -r requirements.txt
```
